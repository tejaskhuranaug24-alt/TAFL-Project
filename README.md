# NFA to DFA Subset Construction Visualizer

This is a static web application that converts an NFA into an equivalent DFA using subset construction. It provides:

- NFA input form with transitions
- DFA generation and transition table
- Step-by-step subset construction visualization
- Graphical rendering of the original NFA and resulting DFA

## Run locally

Open `index.html` in your browser or use a local web server:

```bash
python -m http.server
```

Then visit `http://localhost:8000`.
